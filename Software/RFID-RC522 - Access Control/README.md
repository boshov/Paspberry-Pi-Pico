## Wiring
| RFID-RC522  | Raspberry Pi Pico |
| ------------- | ------------- |
| 3.3V  | 3.3V |
| RST  | GPIO 22 |
| GND  | GND |
| IRQ  | - |
| MISO  | GPIO 16 |
| MOSI  | GPIO 19 |
| SCK  | GPIO 18 |
| SDA  | GPIO 17|

| OLED display  | Raspberry Pi Pico |
| ------------- | ------------- |
| VCC  | 3.3V |
| GND  | GND |
| SCL  | GPIO 5 |
| SDA  | GPIO 4 |
