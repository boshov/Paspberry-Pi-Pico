## Wiring
| Epaper display  | Raspberry Pi Pico |
| ------------- | ------------- |
| 3.3V  | 3.3V |
| GND  | GND |
| SDI  | GPIO 19 |
| SCK  | GPIO 18 |
| CS  | GPIO 5 |
| D/C  | GPIO 8 |
| RES  | GPIO 9 |
| BUSY  | GPIO 7 |
